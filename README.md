## 開発経緯 
友人が美容室の経営をしており、自店舗で使用できるポイント加算アプリを試しに作成してほしいという要望で作成しました。

## アプリの要約
追加用のQRコードを読み取ることで、ポイントが加算するようにしました。
また、たまったポイントで商品と交換できる機能も作成しました。
商品の交換は、交換したい商品のQRコードを読み取ることでポイントと交換できます。
